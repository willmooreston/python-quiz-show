# python-quiz-show

Flashcard-style interview drill app built with React + Vite. Supports multiple choice and fill-in-the-blank questions, tracks misses, and lets you drill only the ones you got wrong.

**Live:** https://willmooreston.github.io/python-quiz-show/

## Quizzes

| Quiz | Topics |
|------|--------|
| Python Basics | Built-ins, Slicing, Strings, Dicts, Comprehensions |

## Stack

- React 19
- Vite 8
- GitHub Actions + GitHub Pages

## Running locally

```
npm install
npm run dev
```
